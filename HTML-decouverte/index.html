Bienvenue sur le site d'entraînement HTML pour étudiants. Vous trouverez ici des exemples de syntaxe HTML pour vous entraîner à la création de pages web.
Le site est divisé en plusieurs pages pour vous permettre de vous exercer sur différents types de balises et de structures.

HTML (HyperText Markup Language) est un langage de balisage utilisé pour structurer le contenu d'une page web. Il est composé de balises qui définissent la structure et le contenu de la page.
CSS(Cascading Style Sheets) est un langage de feuilles de style utilisé pour définir la présentation et la mise en forme CSS des éléments HTML d'une page web.
