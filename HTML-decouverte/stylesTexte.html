Exemples de Mise en forme CSS de texte

Pour mettre en forme du texte, on peut utiliser les balises <span> et les attributs style ou class pour appliquer des styles CSS.

  • La propriété css style="text-decoration: underline" va permettre de souligner un texte.
  • La propriété css style="text-transform: uppercase" va permettre de mettre en majuscules un texte.
  • La propriété css style="text-transform: lowercase" va permettre de mettre en minuscules un texte.
  • La propriété css style="text-transform: capitalize" va permettre de mettre en majuscule la première lettre de chaque mot d'un texte.
  • La propriété css style="text-align: center" va permettre de centrer un texte.
  • La propriété css style="font-style: italic" va permettre de mettre en italique un texte.
  • La propriété css style="font-weight: bold" va permettre de mettre en gras un texte.
  • La propriété css style="font-size: 1.5em" va permettre de changer la taille de la police d'un texte.
  • La propriété css style="font-family: 'Courier New', monospace" va permettre de changer la famille de police d'un texte.
  • La propriété css style="color: red" va permettre de changer la couleur du texte.
  • La propriété css style="text-decoration: line-through" va permettre de barrer un texte.
