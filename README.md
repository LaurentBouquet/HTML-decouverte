## Lancer un Codespace

1. Ouvrir le terminal de VSCode.
2. Dans le terminal, taper la commande :
    ```sh
    npm i -g http-server
    ```
3. Dans le terminal, taper la commande :
    ```sh
    http-server
    ```
4. Dans la fenêtre de prompt, cliquer sur "Ouvrir dans un navigateur".
